# Документация Payment Transactions System

## UML диаграммы

### Use Case диаграмма (Диаграмма вариантов использования)

Файл: `use-case-diagram.puml`

Диаграмма показывает взаимодействие актеров (пользователей) с системой управления платежными транзакциями.

#### Акторы

1. **Пользователь (User)** - обычный зарегистрированный пользователь системы
2. **Администратор (Admin)** - пользователь с расширенными правами (наследует все права User)

#### Основные группы функций

##### 1. Аутентификация
- Регистрация нового пользователя
- Вход в систему
- Выход из системы

##### 2. Управление транзакциями
- Просмотр списка транзакций
- Создание новой транзакции (доход/расход)
- Редактирование существующей транзакции
- Удаление транзакции
- Фильтрация по типу, статусу, категории
- Просмотр истории изменений транзакции

##### 3. Аналитика и отчеты
- Просмотр дашборда с общей статистикой
- Графики доходов и расходов
- Аналитика по категориям
- Месячная статистика за 6 месяцев
- Топ-10 категорий расходов
- Последняя активность

##### 4. Управление категориями
- Просмотр всех категорий
- Создание новой категории с цветом и иконкой
- Редактирование категории
- Удаление категории

##### 5. Администрирование (только Admin)
- Просмотр всех пользователей системы
- Редактирование данных пользователя
- Удаление пользователя
- Просмотр статистики системы
- Просмотр транзакций, ожидающих подтверждения
- Одобрение транзакции (статус → completed)
- Отклонение транзакции (статус → failed)

##### 6. Помощь
- Просмотр руководства пользователя

#### Типы отношений

- **Наследование (Generalization)**: Администратор наследует все права Пользователя
- **Include**: Обязательные связи (например, просмотр транзакций включает фильтрацию)
- **Extend**: Опциональные расширения (одобрение/отклонение расширяют просмотр ожидающих транзакций)

## Как просмотреть диаграмму

### Способ 1: Онлайн редактор PlantUML

1. Перейдите на [PlantUML Online Editor](http://www.plantuml.com/plantuml/uml/)
2. Скопируйте содержимое файла `use-case-diagram.puml`
3. Вставьте в редактор
4. Диаграмма отобразится автоматически

### Способ 2: VS Code расширение

1. Установите расширение "PlantUML" в VS Code
2. Откройте файл `use-case-diagram.puml`
3. Нажмите `Alt+D` для предварительного просмотра

### Способ 3: PlantUML локально

```bash
# Установка PlantUML
sudo apt-get install plantuml  # Linux
brew install plantuml          # macOS

# Генерация PNG изображения
plantuml docs/use-case-diagram.puml

# Генерация SVG
plantuml -tsvg docs/use-case-diagram.puml
```

### Способ 4: GitHub (автоматический рендеринг)

GitHub не рендерит PlantUML диаграммы напрямую, но можно использовать proxy:

```markdown
![Use Case Diagram](http://www.plantuml.com/plantuml/proxy?src=https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/docs/use-case-diagram.puml)
```
