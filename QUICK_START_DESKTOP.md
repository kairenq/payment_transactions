# 🖥️ Десктопное приложение - Быстрый старт

## 🎯 Что получишь?

Приложение открывается **в отдельном окне** (не в браузере), как обычная программа!

---

## ⚡ Вариант 1: Запустить сразу (без .exe)

### Windows:
1. Двойной клик на: **`start-desktop-app.bat`**
2. Готово! Приложение откроется в отдельном окне

### Требования:
- Node.js 18+: https://nodejs.org/
- Python 3.8+: https://www.python.org/

**Первый запуск**: установка зависимостей ~2-3 минуты
**Следующие запуски**: сразу открывается (~5 секунд)

---

## 📦 Вариант 2: Создать .exe инсталлятор

Если хочешь установить на другие компьютеры:

### Windows:
1. Двойной клик на: **`build-desktop-app.bat`**
2. Жди 5-10 минут (сборка)
3. Готово! Файл здесь: `electron\dist\Payment Transactions Setup 1.0.0.exe`

### Установка на другом компе:
1. Скопируй `.exe` файл
2. Двойной клик → установка
3. Ярлык на рабочем столе → запуск!

## 🛠️ Кастомизация

### Изменить иконку приложения:
1. Создай `icon.png` (256x256 или 512x512 пикселей)
2. Положи в папку `electron/`
3. Пересобери `.exe` через `build-desktop-app.bat`

### Изменить название окна:
Открой `electron/main.js`, найди:
```javascript
title: 'Payment Transactions System'
```
Измени на своё название.
