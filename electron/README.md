# Electron Desktop App

## Quick Start

### Windows
Double-click: `start-desktop-app.bat`

### Linux/Mac
```bash
chmod +x start-desktop-app.sh
./start-desktop-app.sh
```

## Build .exe installer

### Windows
Double-click: `build-desktop-app.bat`

Result: `electron/dist/Payment Transactions Setup 1.0.0.exe`

## Custom Icon

Replace `electron/icon.png` with your own icon (256x256 or 512x512 pixels).
